# Changelog

All notable changes to this project are documented in this file.

## [0.0.1] - 2026-04-12

### Added
- `examples/bboxes/` with reusable GeoJSON inputs for documented workflows.
- Initial release tag and packaging metadata for `relievo`.

### Changed
- Project rebranded from `blender-relief` to `relievo` across package, CLI, docs, tests, and CI.
- Python package path renamed from `blender_relief/` to `relievo/`.
- Documentation and script examples updated to use the `relievo` command and new example paths.

### Fixed
- CI lint target paths updated to match the renamed package.
